# OPQBot-GroupManager
OPQBot 群管理机器人
目前处于测试状态，软件不一定稳定
# 使用教程
1. 下载编译好的二进制文件
2. 将`config.yaml.example`内需要更改的配置项填写好并重命名为`config.yaml`, 启动程序即可😊
